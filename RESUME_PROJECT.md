# 简历项目介绍

## Vue2 TodoList - 日常任务管理应用

### 项目简介
基于 Vue.js Options API 开发的任务管理单页应用，支持今日/明日任务分类管理，具备完整的 CRUD 功能和数据持久化能力。采用前后端分离架构，后端使用 Express + JSON 文件存储数据。

### 技术栈
- **前端框架**: Vue 3 (Options API 风格)
- **构建工具**: Vite
- **后端框架**: Express.js
- **数据存储**: JSON 文件 (tasks.json)
- **样式方案**: CSS3 + Flexbox 响应式布局 + 网格背景

### 核心功能
- 任务增删改查（CRUD）
- 任务完成状态切换
- 今日/明日任务分类管理
- 实时任务统计
- 后端数据持久化
- 响应式设计适配移动端
- 数据自动迁移（localStorage → 后端）

### 技术亮点
1. **组件化架构**: 采用单一职责原则，拆分为 Header、Input、List、Item 四个独立组件
2. **前后端分离**: 使用 REST API 实现前后端通信
3. **数据持久化**: Express 服务器 + JSON 文件存储
4. **父子组件通信**: Props 向下传递数据，$emit 向上传递事件
5. **响应式数据管理**: 使用 computed 计算属性实现任务统计
6. **列表动画**: 使用 transition-group 实现任务添加/删除的平滑过渡效果
7. **用户体验优化**: 空状态提示、悬停交互反馈、键盘回车快捷添加

### 项目收获
- 深入理解 Vue Options API 的组件开发模式
- 掌握组件通信、计算属性、侦听器等核心概念
- 实践前端组件化设计思想和单向数据流架构
- 学习 Node.js/Express 后端开发基础
- 理解前后端分离架构和 REST API 设计
